Static site builder¶

Static site generator

Requirements: lux.extensions.base

Usage¶

Put the lux.extensions.static extension into your EXTENSIONS list and build the static web site via the build_static option in the command line:

python managet.py build_static

Parameters¶

EXTRA_FILES¶

Default: '()'

List/tuple of additional files to copy to the STATIC_LOCATION

MD_EXTENSIONS¶

Default: "['extra', 'meta']"

List/tuple of makrdown extensions

METADATA_PROCESSORS¶

Default: '[]'

A list of functions to perocess metadata

RELATIVE_URLS¶

Default: 'False'

Display urls as relative paths (useful during development)

SNIPPETS_LOCATION¶

Default: 'snippets'

Directory where to find snippets used as content

SOURCE_SUFFIX¶

Default: 'md'

The default suffix of source filenames

STATIC_LOCATION¶

Default: 'build'

Directory where the static site is created

STATIC_SITEMAP¶

Default: '{}'

Dictionary of contents for the site

STATIC_TEMPLATE¶

Default: 'PageTemplate(div)'

Default static template
