Social Integration and OAuth¶

This extension facilitates login via third party authentication providers using either the OAuth1 or OAuth2 protocol. In addition, it allows the use of the Open Graph Protocol (OGP) for social graphs and twitter cards and additional APIs a provider may have (google analytics, google map for example).

Usage¶

Add lux.extensions.oauth to the EXTENSIONS list of your project and include the OAUTH_PROVIDERS dictionary:

OAUTH_PROVIDERS = {
    'google': ...,
    'twitter': ...,
}

This extension adds open graph meta tags when the DEFAULT_OG_TYPE is not None (it is set to website by default).
