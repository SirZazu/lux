Base Extension¶

The base lux extension provides several middleware utilities. If used, it should be the first extension in your EXTENSIONS list (excluding extensions which don’t provide any wsgi middleware such as lux.extensions.sitemap for example).

Media Handling¶

When the SERVE_STATIC_FILES parameter is set to True, this extension adds middleware for serving static files from MEDIA_URL. In addition, a FAVICON location can also be specified.
