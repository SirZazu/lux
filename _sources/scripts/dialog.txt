.. _js-dialog:

=================
Dialog
=================

To create a dialog on the fly::

    var dialog = lux.web.dialog({closable: true});


Available Options
=========================

.. _js-dialog-dragdrop:

dragdrop
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

An optional :ref:`Drag and drop object <js-dnd>` to manage drag and drop
functionalities.
If this is given the dialog will be draggable and it can be dropped into
other containers.
