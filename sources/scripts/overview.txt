==========================
Overview
==========================

The javascript code create a single variables ``lux`` stored in the ``window``
global object.

.. _js-dependencies:

Dependencies
==================
There are two dependencies for the core web library:

* requirejs_
* jQuery_


Build
======
To build the javascript code you need grunt_, a
command line build tool for javascript projects. On the distribution directory
type::

    grunt -h
    
for help on usage.





.. _grunt: http://gruntjs.com/
.. _requirejs: http://requirejs.org/
.. _jQuery: http://jquery.com/
