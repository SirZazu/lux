Writing Extensions¶

Extension are implemented by subclassing the Extension class:

import lux

class MyExtension(lux.Extension):
    # Optional version number
    _version = '0.1.0'

    def middleware(self, app):
        ...

The Extension.middleware() method is called once only by the App serving the web applications, and it must return an iterable over WSGI middleware or None.

Events¶

An Extension can register several callbacks which are invoked at different points during the application live-span. These callbacks receive as first positional argument, the App instance running the web site and are implemented by adding some of the following methods to your extension class:

on_config¶

Extension.on_config(self, app)¶

This is the first event to be fired. It is executed once only after the App.config dictionary has been loaded from the setting file. This is a chance to perform post processing on parameters before the wsgi App.handler is loaded.

on_loaded¶

Extension.on_loaded(self, app, handler)¶

Called once only when all Extension have loaded their middleware() into the WSGI handler. A chance to add additional middleware or perform any sort of post-processing on the wsgi application handler.

on_start¶

Extension.on_start(self, app, server)¶

Called once only just before the pulsar server is about to start serving the app.

on_request¶

Extension.on_request(self, app, request)¶

Called when a new request is received by the App instance. This event occurs before the application loops through the WSGI middleware to produce the response.

on_html_document¶

Extension.on_html_document(self, app, request, doc)¶

Called the first time the request.html_document attribute is accessed. A chance to add static data or any other Html specific information.

on_form¶

Extension.on_form(self, app, form)¶

on_html_response¶

Extension.on_html_response(self, app, request, html)¶

Called by the Html.html_response method. The html input is a dictionary containing the body keys with the html element which is being rendered in the body part of the Html page.
