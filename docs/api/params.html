Parameters¶

A full list of available Parameter from all Extension available in the standard lux distribution.

Core¶

APP_NAME¶

Default: 'Lux'

Application site name

ASSET_PROTOCOL¶

Default: ''

Default protocol for scripts and links (could be http: or https:)

CACHE_SERVER¶

Default: 'None'

Cache server, can be a connection string to a valid datastore which support the cache protocol or an object supporting the cache protocol

DATETIME_FORMAT¶

Default: 'd MMM y, h:mm:ss a'

Default formatting for dates in JavaScript

DATE_FORMAT¶

Default: 'd MMM y'

Default formatting for dates in JavaScript

DEFAULT_FROM_EMAIL¶

Default: ''

Default email address to send email from

DEFAULT_TEMPLATE_ENGINE¶

Default: 'python'

Default template engine

DESCRIPTION¶

Default: 'None'

A description to display before the argument help on the command line when running commands

ENCODING¶

Default: 'utf-8'

Default encoding for text.

ERROR_HANDLER¶

Default: 'None'

Handler of Http exceptions

EXTENSIONS¶

Default: '[]'

List of extension names to use in your application. The order matter since the wsgi middleware of extension is processed in the order given by this setting.

HTML_LINKS¶

Default: '[]'

List of links to include in the html head tag.

HTML_META¶

Default: "[{'content': 'IE=edge', 'http-equiv': 'X-UA-Compatible'}, {'content': 'width=device-width, initial-scale=1', 'name': 'viewport'}]"

List of default meta elements to add to the html headelement

HTML_TITLE¶

Default: 'Lux'

Default HTML Title

LINKS¶

Default: "{'python': 'https://www.python.org/', 'pulsar': 'http://pythonhosted.org/pulsar', 'lux': 'https://github.com/quantmind/lux'}"

Links used throughout the web site

LOCALE¶

Default: 'en_GB'

Default locale

LOGGING_CONFIG¶

Default: 'None'

Dictionary for configuring logging

MEDIA_URL¶

Default: '/media/'

the base url for static files

MINIFIED_MEDIA¶

Default: 'True'

Use minified media files. All media files will replace their extensions with .min.ext. For example, javascript links *.js become *.min.js

REQUIREJS¶

Default: "('require',)"

Default Required javascript. Loaded via requirejs.

REQUIREJS_CONFIG¶

Default: 'http://quantmind.github.io/require-config-js/require.config'

Configuration url for Requirejs

REQUIREJS_URL¶

Default: '//cdnjs.cloudflare.com/ajax/libs/require.js/2.1.14/require'

Default url for requirejs

SCRIPTS¶

Default: '[]'

List of scripts to load in the head tag

SITE_URL¶

Default: 'None'

Web site url

Base¶

CLEAN_URL¶

Default: 'False'

When True, requests on urls with consecutive slashes are converted to valid url and redirected.

FAVICON¶

Default: 'None'

Adds tag of type image/x-icon in the head section of the Html document

GZIP_MIN_LENGTH¶

Default: '200'

If a positive integer, a response middleware is added so that it encodes the response via the gzip algorithm.

SERVE_STATIC_FILES¶

Default: 'False'

if True add middleware to serve static files.

USE_ETAGS¶

Default: 'False'

Sessions¶

ACCOUNT_ACTIVATION_DAYS¶

Default: '2'

Number of days the activation code is valid

AUTHENTICATION_BACKEND¶

Default: 'lux.extensions.sessions.permissions.AuthBackend'

Python dotted path to a class used to provide a backend for authentication.

AUTH_SALT_SIZE¶

Default: '8'

Salt size for encription algorithm

CHECK_USERNAME¶

Default: '<function Extension.<lambda> at 0x10569a510>'

Check if the username is valid

CRYPT_ALGORITHM¶

Default: 'lux.utils.crypt.arc4'

Python dotted path to module which provides the encrypt and descript method for password and sensitive data encryption/decription

CSRF_PARAM¶

Default: 'authenticity_token'

CSRF parameter name in forms

DEFAULT_PERMISSION_LEVEL¶

Default: 'read'

When no roles has tested positive for permissions, this parameter is used to check if a model has permission for an action

PERMISSION_LEVELS¶

Default: "{'read': 10, 'delete': 40, 'update': 30, 'create': 20}"

When a model

RESET_PASSWORD_URL¶

Default: ''

If given, add the router to handle password resets

SECRET_KEY¶

Default: 'secret-key'

A string or bytes used for encripting data. Must be unique to the application and long and random enough

SESSION_COOKIE_NAME¶

Default: 'LUX'

Name of the cookie which stores session id

SESSION_EXPIRY¶

Default: '604800'

Expiry for a session in seconds.

API¶

API_DOCS_URL¶

Default: 'api/docs'

API_LIMIT_AUTH¶

Default: '100'

Maximum number of items returned when user is authenticated

API_LIMIT_DEFAULT¶

Default: '30'

Default number of items returned

API_LIMIT_KEY¶

Default: 'limit'

API_LIMIT_NOAUTH¶

Default: '30'

Maximum number of items returned when user is not authenticated

API_OFFSET_KEY¶

Default: 'offset'

API_SEARCH_KEY¶

Default: 'q'

The query key for full text search

API_URL¶

Default: 'api'

DATASTORE¶

Default: 'None'

Dictionary for mapping models to their back-ends database

DUMPDB_EXTENSIONS¶

Default: 'None'

ODM¶

Default: 'None'

Optional Object Data Mapper.

SEARCHENGINE¶

Default: 'None'

Search engine for models

Web design¶

BOOTSTRAP¶

Default: '//maxcdn.bootstrapcdn.com/bootstrap/3.2.0/css/bootstrap'

Twitter bootstrap url. Set to none to not use it.

EXCLUDE_EXTENSIONS_CSS¶

Default: 'None'

Optional list of extensions to exclude form the css

FONTAWESOME¶

Default: '//netdna.bootstrapcdn.com/font-awesome/4.2.0/css/font-awesome'

FONTAWESOME url. Set to none to not use it.

Static Site¶

CONTEXT_LOCATION¶

Default: 'context'

Directory where to find files to populate the context dictionary

MD_EXTENSIONS¶

Default: "['extra', 'meta', 'toc']"

List/tuple of markdown extensions

STATIC_API¶

Default: 'api'

Build a JSON api, required when using angular in Html5 navigation mode

STATIC_LOCATION¶

Default: 'build'

Directory where the static site is created

STATIC_MEDIA¶

Default: 'True'

Add handler for media files

STATIC_SPECIALS¶

Default: "('404',)"

paths included in this list won’t create the json api

STATIC_TEMPLATE¶

Default: 'home.html'

Default static template
