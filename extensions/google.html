Google integration¶

Facilitate google integration

Google API Key¶

If the GOOGLE_API_KEY parameter is set to a valid google api key, this extension allows to interact with several google services.

Google maps¶

To use google maps api, you can require it via the require() method:

>>> doc.head.scripts.require('google-maps')

Google Analytics¶

Set the GOOGLE_ANALYTICS_ID parameter to a valid analytic ID.
