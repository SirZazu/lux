Web Extensions¶

Extension class¶

Extension.destroy(instance)¶

Destroy the instance, including the instance.placeholder().

Extension.instance(elem)¶

Try to find an instance of this Extension from elem.

Arguments:
  • elem – an extension instance, and Html element or a jQuery dom element.
Returns:

The closest instance to elem or nothing.

Extension instance¶

An extension instance o has the following base API:

instance.id()¶
Returns:unique string id for the instance. The id is stored in the instance.element() data.
instance.element()¶
Returns:The jQuery dom element managed by the instance.
instance.destroy()¶

Destroy the instance, Proxy to Extension.destroy()
