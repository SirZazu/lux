Documentation for lux's DEVELOPMENT version. Get the release docs here.

Overview¶

The javascript code create a single variables lux stored in the window global object.

Dependencies¶

There are two dependencies for the core web library:

  • requirejs
  • jQuery

Build¶

To build the javascript code you need grunt, a command line build tool for javascript projects. On the distribution directory type:

grunt -h

for help on usage.
