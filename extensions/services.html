API¶

api_function¶

class lux.extensions.services.binder.api_function(path=None, nargs=0, allowed_params=None, required_params=None, callback=None, method=None, append=None, doc=None, **http_params)[source]¶

Bind an API function to an API.

name¶

This attribute is set at runtime by the API metaclass.

It is the declarative name of this api_function in a API class.

path¶

Relative or absolute url of api function.

If a relative url, the full url is constructed by prepending the the API.BASE_URL

nargs¶

Number of required arguments used to form the final url. It the value is set to “*”, the number can vary and no check is performed.

required_params¶

Tuple of parameter names required (not optional) by the api call.

Can be None.

http_params¶

Additional parameters to pass to the http request method.
